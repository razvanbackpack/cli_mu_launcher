## Features

![Settings Menu](repo_assets/Mu%20Launcher%20Settings.png)

![Resolution Selection](repo_assets/Mu%20Launcher%20Resolution.png)

## Quick Start

1. Build the launcher
2. Place in your game folder
3. Run and configure settings
4. Click "PLAY GAME"
